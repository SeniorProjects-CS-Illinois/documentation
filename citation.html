Citation

Usage

  1. Selecting the type of item to cite

    Select the item type using the spinner at the top of the screen. The fields displayed below it and the format of the generated citation will change depending on the item type.
    (Pic of combo box, pic of items in it)

  2. Populate the citation fields

    Automatically - use the item view

    Press a button on the item view page to automatically populate the citation fields with information from the VuFind catalog

    (Pic of item view page, with cite button highlighted/selected/circled)

    Manually

    Click on each field and manually enter in the information to be used in the citation. Each empty field displays what type of information should be entered into it in greyed out text.

    (Pic of empty citation fields)

  3. Generate the citation

    Select the citation type you need by touching the radio button next to the corresponding type:

    (Pic of radio buttons/selector)

    Touch the “generate citation” button to generate the citation; the generated citation will be displayed in a pop-up dialog.

    (Pic of pop-up dialog)

    To use the citation, either long-press it and copy it to your device’s clipboard, or press the “E-mail citation” button to email it.

Description

  • Fields for citation

    These will change depending on what citation type was selected on the previous screen (We may need to describe each field for each type, or something along those lines - it might help)

  • Citation types

    • APA - American Psychological Association - http://www.apa.org/
    • MLA - Modern Language Association - http://www.mla.org/

  • Email citation

    Send an email containing the generated citation using an email client on the phone (a popup dialog will let you select which one if your system has more than one installed)
