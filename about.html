About

Citation Module

Purpose:

The citation module is designed to allow a user to quickly and accurately cite a wide variety of sources in either MLA or APA format. The module also allows the user to email the completed citation.

Login (VuFind) Module

Purpose:

The Login Module will display all of the information associated with a user’s vufind/I-share library account. Some categories of information (Favorites, Checked-out, Requested) allow the user to interact with the information, by renewing a checked-out item, cancelling a requested item, or adding items to/removing items from a favorites list, as well as sharing the favorites to Facebook.
